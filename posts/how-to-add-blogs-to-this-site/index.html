Lets get started!!!

Required softwares

First things first !.. Lets see the list of all the required softwares.

  • Hugo is the framework which we are using to generate the static HTML from the markdown which we are writing.
  • Visual Studio Code for editing the markdown files and to push the chnages to the git

Install Hugo

Official page for installing hugo is here Here we will see the installation instructions for windows only because i assume thats the most used operation system. You can use the operation systen of your choice.

I suggest the windows users to install packages via chocolatey. The package manager for windows(Like apt-get or yum for windows).

  • To install chocolatey, you can follow this link for full instructions or open an administrative powershell and copy paste the below command.
Set-ExecutionPolicy Bypass -Scope Process -Force; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))
